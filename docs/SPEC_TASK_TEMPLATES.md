# タスクテンプレート機能 仕様書

**作成日**: 2026-03-25

---

## 1. 概要

タスクの種類（担当ロール）に応じた依頼テンプレートを定義し、タスク依頼時にテンプレートを選択すると、フォームフィールドが動的に変更される機能。

## 2. テンプレート一覧（初期）

### 2.1 バナー・エキスパート
```yaml
名前: バナー制作
カテゴリ: デザイン
デフォルトフィールド:
  - バナーサイズ (選択: 1200x628, 1080x1080, 728x90, カスタム)
  - 用途 (選択: Web広告, SNS, メルマガ, LP, その他)
  - トーン・テイスト (テキスト: 例: 明るく親しみやすい)
  - 参考デザインURL (URL)
  - 素材提供 (選択: あり, なし, 一部あり)
  - テキスト原稿 (テキストエリア)
  - 納品形式 (選択: PSD, PNG, JPG, SVG, Figma)
```

## 3. データモデル

### 3.1 task_templates テーブル
```sql
CREATE TABLE task_templates (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  category TEXT NOT NULL,
  fields JSONB NOT NULL DEFAULT '[]',
  is_default BOOLEAN DEFAULT false,
  created_by UUID REFERENCES users(id),
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
```

### 3.2 fields JSONB構造
```json
[
  {
    "key": "banner_size",
    "label": "バナーサイズ",
    "type": "select",
    "options": ["1200x628", "1080x1080", "728x90", "カスタム"],
    "required": true
  },
  {
    "key": "tone",
    "label": "トーン・テイスト",
    "type": "text",
    "placeholder": "例: 明るく親しみやすい",
    "required": false
  }
]
```

### 3.3 タスクへの紐付け
- tasks テーブルに `template_id UUID` と `template_data JSONB` を追加
- `template_data`: テンプレートフィールドに入力された値を保存

## 4. 画面仕様

### 4.1 タスク依頼 Step1 の変更
- フォーム最上部に「テンプレート選択」ドロップダウンを追加
- テンプレート選択時: 共通フィールド（クライアント名、タスク名等）の下に、テンプレート固有フィールドが動的に表示
- 「テンプレートなし（汎用）」オプションも選択可能

### 4.2 テンプレート管理画面 (/settings → テンプレートタブ or /templates)
- テンプレート一覧: 名前、カテゴリ、フィールド数、デフォルト/カスタム
- 新規作成: 名前、カテゴリ、フィールド定義（動的追加）
- 編集: 既存テンプレートのフィールド追加・削除・並べ替え
- 削除: カスタムテンプレートのみ削除可（デフォルトは削除不可）
- フィールド型: text, textarea, select, number, url, multiselect

### 4.3 タスク詳細の変更
- テンプレートデータがある場合、依頼情報カードにテンプレートフィールドの値を表示
