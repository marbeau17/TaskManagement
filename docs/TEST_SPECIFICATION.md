# テスト仕様書 — WorkFlow タスク管理ツール

## テスト方針
- E2Eテスト（Playwright）で本番URLに対してGUI操作を検証
- モンキーテスト要素: 各画面で予期しない操作・空入力・境界値を試行
- 統合テスト: 画面間のデータ連携を検証

## テストケース一覧

### TC-01: ログイン画面
- [ ] ログイン画面が表示される
- [ ] 空メール/空パスワードでsubmitできない（バリデーション）
- [ ] 不正な認証情報でエラーメッセージ表示
- [ ] 正しい認証情報でダッシュボードに遷移
- [ ] 「ログイン情報を自動入力」ボタンでフィールドが埋まる

### TC-02: ダッシュボード
- [ ] サイドバーが表示される
- [ ] KPIカードが4つ表示される
- [ ] クリエイター別/クライアント別タブが切替可能
- [ ] 期間切替（今週/今月/全期間）が動作
- [ ] 「+ タスク依頼」ボタンが遷移する

### TC-03: タスク一覧
- [ ] タスクテーブルが表示される
- [ ] ステータスタブ（すべて/アサイン待ち等）が動作
- [ ] タスク行クリックで詳細に遷移

### TC-04: タスク依頼（作成）
- [ ] Step1フォームが表示される
- [ ] 必須フィールド未入力でバリデーションエラー
- [ ] Step2（アサイン）に遷移可能

### TC-05: タスク詳細
- [ ] タスク情報が表示される
- [ ] 進捗バーとボタンが表示される
- [ ] コメント欄が表示される
- [ ] 「アサイン変更」ボタンが存在する

### TC-06: 稼働管理
- [ ] KPIカードが表示される
- [ ] メンバー稼働テーブルが表示される

### TC-07: メンバー管理
- [ ] メンバー一覧が表示される
- [ ] 「メンバー一覧」「組織図」タブ切替
- [ ] 組織図が表示される
- [ ] 編集モーダルが開く
- [ ] 名前変更が可能

### TC-08: クライアント管理
- [ ] クライアント一覧が表示される
- [ ] 追加ボタンでモーダル表示
- [ ] クライアント名入力・作成

### TC-09: 設定
- [ ] 設定画面が表示される
- [ ] テーマ切替が存在する

### TC-10: ダークモード
- [ ] テーマ切替でダークモードに変更可能
- [ ] ダークモードのCSS変数が適用される

### TC-11: ナビゲーション
- [ ] 全サイドバーリンクが正しく遷移する
- [ ] ログアウトが動作する
