# 仕様書: パイプライン管理テーブルフィルター機能

## 概要
パイプライン管理の案件一覧テーブルに、クライアント名・紹介先・案件名・状況の4項目でフィルタリングできる機能を追加する。

## フィルター仕様

| フィルター | 種別 | 動作 |
|---|---|---|
| クライアント名 | テキスト入力 | 部分一致（大文字小文字無視） |
| 紹介先 | セレクト | 一致するもののみ表示（ユニーク値リスト） |
| 案件名 | テキスト入力 | 部分一致（大文字小文字無視） |
| 状況 | セレクト | Firm / Namelikly / Win / Lost から選択 |

## UI配置
- テーブルヘッダー行の直下にフィルター行を追加
- 各フィルターは対応する列の位置に配置
- 「クリア」ボタンで全フィルターをリセット

## 動作フロー
1. フィルター入力/選択するとリアルタイムでテーブルが絞り込まれる
2. ソート機能と併用可能（フィルター後のデータに対してソート）
3. CSV出力はフィルター適用後のデータを対象とする
4. 売上サマリータブには影響しない
